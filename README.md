# 🧹 SJC Ultimate Cleaner

**SJC Ultimate Cleaner** — это мощная утилита для комплексной очистки системы, управления Steam и запуска OldRust в безопасном режиме с эмуляцией HWID.

Разработано для опытных пользователей, которые ценят контроль над своей системой.

## 🚀 Возможности

### Основные функции

- **BYPASS (Полная очистка)**
  - Закрытие процессов Steam
  - Очистка реестра Rust (Facepunch, OldDevblog, MYRUST240)
  - Очистка истории Win+R
  - Очистка истории браузера (rundll32)
  - Удаление `.dll` записей из FeatureUsage
  - Удаление `ssfn*` файлов Steam
  - Сброс `loginusers.vdf`
  - Удаление `avatarcache` и `userdata`
  - Удаление `coplay_*.vdf`
  - Очистка реестра Steam (AutoLoginUser и др.)
  - Сброс сетевой статистики (SRU)
  - Очистка TEMP папок
  - Удаление папки `C:\OldRust`
  - Удаление `EasyAntiCheat.dll` с рабочего стола
  - Очистка Prefetch от следов `extreme` и `injector`

- **RUST Manager**
  - Запуск OldRust с мониторингом DLL (фильтр системных папок)
  - Резервное копирование всех DLL
  - Интерактивное удаление новых DLL
  - Удаление EasyAntiCheat.dll

- **Безопасный запуск OldRust**
  - Генерация фейкового HWID
  - Скрытый запуск процесса
  - Изменение переменных окружения (COMPUTERNAME, USERNAME)
  - Частичное скрытие от античитов
