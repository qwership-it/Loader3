# 🧹 SJC Ultimate Cleaner

**SJC Ultimate Cleaner** — консольная утилита для очистки системы, управления Steam и безопасного запуска OldRust с эмуляцией HWID.

Проект представляет собой **один файл** `main.cpp` (около 700 строк) на C++17, использующий только стандартные WinAPI.

## 🚀 Возможности

### Главное меню
- **[1] CLEAR (FULL CLEAN)** — полная очистка системы (15 шагов).
- **[2] OLDRUST (Manager)** — подменю для работы с OldRust.
- **[0] EXIT** — выход.

### Меню OLDRUST
- **[1] Run OldRust with HWID Spoof** — запускает `RustClient.exe` скрыто с эмуляцией HWID, подменой переменных `COMPUTERNAME`, `USERNAME` и записью фейкового GUID в реестр.
- **[2] Delete EasyAntiCheat.dll** — ищет EAC по 8 популярным путям и удаляет (с запросом подтверждения).
- **[3] Show DLL list** — открывает PowerShell с таблицей загруженных DLL для процесса `RustClient.exe` (фильтр: скрывает системные папки).

### Очистка BYPASS (15 шагов)
1. Закрытие Steam (`steam.exe`, `steamwebhelper.exe`).
2. Удаление ключей реестра Rust (`Facepunch Studios LTD`, `OldDevblogRust`, `MYRUST240DEVBLOG`).
3. Очистка истории Win+R.
4. Очистка истории браузера (через `rundll32`).
5. Очистка `FeatureUsage\AppSwitched`.
6. Удаление `ssfn*` файлов Steam.
7. Сброс `loginusers.vdf`.
8. Удаление `avatarcache` и всех папок в `userdata`.
9. Удаление `coplay_*.vdf`.
10. Очистка реестра Steam (`AutoLoginUser`, `RememberPassword`, `MostRecentUser`).
11. Очистка статистики сети (SRU).
12. Очистка TEMP папок (пользовательской и системной).
13. Удаление папки `C:\OldRust`.
14. Удаление `EasyAntiCheat.dll` с рабочего стола.
15. Очистка Prefetch от файлов, содержащих `extreme` или `injector`.

После очистки Steam автоматически перезапускается.
